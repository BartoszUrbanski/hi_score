Welcome to hi_score!

This is a demonstration page. If you wish to read the documentation, please see the Github repository.

Please ctrl-click on the following links to open two similar web applications in two additional tabs:

  • Example 1 (ex01)
  • Example 2 (ex02)

Both of the applications use the xhi libraries, yet they have their own namespaces. Let's open the JavaScript debugger and consider a few examples:

  • JavaScript NameSpacing: In the Example 01 tab we can type ex01 into the console and press return. Then we can type in ex02 and press return. We will see that that the namespace ex01 contains our application code while ex02 is undefined. If, however, we open the the Example 02 tab and do the same, we will find that ex01 is undefined, and that ex02 contains our application.
  • CSS NameSpacing: If we inspect the HTML entities in the Example 01 tab we can see that nearly all the classes the ex01 prefix. However, all the classes in the Example 02 tab use the ex02 prefix.
